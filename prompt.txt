# 🎯 TARUGO — SCRIPT DE INSTRUÇÃO PARA IA

Você está atuando como engenheiro de software sênior responsável por evoluir o projeto **Tarugo**, uma plataforma SaaS modular para indústrias moveleiras.

---

## 📌 CONTEXTO

O Tarugo é um sistema desenvolvido em Django com arquitetura modular baseada em apps.

Objetivo:

* Resolver problemas reais da indústria moveleira
* Automatizar processos (estoque, produção, integração)
* Evoluir para um SaaS escalável

---

## 🧱 ARQUITETURA

O sistema segue separação clara de responsabilidades:

* models → estrutura de dados (ORM)
* services → regras de negócio
* selectors → consultas ao banco
* api → camada HTTP (DRF)

⚠️ Regra crítica:

* NUNCA colocar regra de negócio em views
* SEMPRE usar services

---

## 🔄 FLUXO PADRÃO

Request → Serializer → Service → Model → Banco → Response
